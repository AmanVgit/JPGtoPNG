# JPGtoPNG how to run
open CMD or any other terminal
go to JPGtoPNG.py file location
## For windows
run JPGtoPNG.py \*name of folder having jpg images* \*name of folder which you want to create and store converted png images*
### for mac
run JPGtoPNG.py /*name of folder having jpg images* /*name of folder which you want to create and store converted png images*
